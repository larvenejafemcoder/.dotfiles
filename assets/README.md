# assets

Supporting data files and scripts for the dotfiles ecosystem.

- `ci/` — CI/CD workflows, Dockerfiles, Brewfiles
- `csv/` — Package lists in CSV format
- `tsv/` — Package lists in TSV format
- `txt/` — Package lists in plain text
- `devbox/` — Devbox/devcontainer provisioning configs
- `scripts/` — Utility scripts (completions, desktop installers, tests)
- `images/` — Image assets (placeholder)
- `md/` — Reference markdown docs (tokei, VIA/QMK, zsh aliases)
