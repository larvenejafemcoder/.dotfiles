# Scripts

Utility scripts for maintenance, testing, and automation.

- `main.sh` — Main dots utility helper
- `dots-test.sh` / `dots-test.bats` — Shell test suite
- `update-packages.sh` — Syncs package lists
- `completions/` — Zsh completions for dots commands
- `desktop/` — Desktop setup scripts (GNOME settings, keybindings, app installers)
- `make-alias-table.sh` — Generates alias reference tables
- `tokei-md-table.sh` — Generates tokei code stats in markdown
- `setup-termux.sh` — Termux bootstrap
- `docker-test.sh` — Docker-based CI test runner
- `git-tag.sh` — Git tag management
- `convert-md-links.sh` — Converts markdown link formats
