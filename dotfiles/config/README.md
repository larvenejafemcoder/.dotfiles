# config

Configuration files for all managed tools. Each subdirectory is a stow package
that gets symlinked into `~/.config/` or `$HOME` by `install.sh`.

Includes: alacritty, autostart, bash, bat, bottom, Code (VS Code), copyq, eza,
fastfetch, ghostty, git, gitui, gtk-3.0, lazydocker, lazygit, lsd, mise, nvim,
ripgrep, ssh, starship, termux, tmux, ulauncher, yazi, zed, zellij, zsh.
