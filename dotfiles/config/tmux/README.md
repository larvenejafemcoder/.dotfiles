# Tmux

Config for tmux — terminal multiplexer.

- `tmux.conf` — Main config (imports modular confs)
- `conf/base.tmux` — Base settings, prefix key, mouse support
- `conf/bind.tmux` — Custom keybindings
- `conf/env.tmux` — Environment variables
- `conf/plugin.tmux` — TPM plugin manager config
- `conf/style.tmux` — Status bar, colors, theme
- `script/` — Utility scripts (battery, window status, session renumbering, hooks)
