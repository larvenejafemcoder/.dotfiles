# Neovim

Config for Neovim — a hyperextensible Vim-based text editor.

- `lazy-lock.json` — Plugin lockfile for lazy.nvim
- `lazyvim.json` — LazyVim distribution settings
- `lua/config/` — Core options and keymaps
- `lua/plugins/` — Plugin configs (blink, colorscheme, lazygit, neo-tree, nvim-cmp)
- `lua/templates/` — Color scheme templates (catppuccin, sonokai, tokyo-night)
