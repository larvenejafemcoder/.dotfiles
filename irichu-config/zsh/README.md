# Zsh

Modular Zsh configuration.

- `.zshrc` — Entry point (sources aliases and rc modules)
- `.zshenv` — Environment variables for all shell types
- `aliases/` — Ordered alias files (hash, suffix, global, general, compdef)
- `completions/` — Custom Zsh completions (`_dots`)
- `rc/` — Modular init scripts (path, env, setopt, history, zstyle, functions, bindkey)
- `rc/libs/` — Third-party integrations (brew, devbox, fzf, zoxide, starship, atuin, nvm, fnm, mise)
