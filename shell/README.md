# Shell

Setup scripts for shell environments and ricing.

- `zsh_setup.sh` — Installs Oh My Zsh + Powerlevel10k theme + Fira Code Nerd Font
- `starship_setup.sh` — Installs Starship binary + generates catppuccin-powerline config
- `rice.sh` — Gruvbox ricing: GTK theme, GNOME extensions, icons, cursors, wallpaper, fish/OMF
