# stow

Stow-compatible config packages. Each subdirectory is a stow package that
can be linked into `$HOME` via `stow -t ~ <package>` or the `deploy.sh` script.

- **alacritty/** — Alacritty terminal config (primary terminal)
- **bash/** — Bash `.bashrc`
- **fastfetch/** — Fastfetch system info
- **fish/** — Fish shell config (SSH keychain)
- **kitty/** — Kitty terminal config (alternative terminal)
- **neofetch/** — Neofetch config + profile images
- **starship/** — Starship prompt (catppuccin-powerline)
- **zsh/** — Zsh `.zshrc` + aliases
