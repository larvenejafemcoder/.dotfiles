# Alacritty

Primary terminal emulator config.

- `alacritty.toml` — Main config (Nordic theme, Meslo font, 0.9 opacity, no decorations)
- `nordic.toml` — Nordic color scheme
- `dank-theme.toml` — Dark purple/black alternate theme
- `keybinds.toml` — Custom keybindings (super+c/v, scroll, font size)
- `alacritty.yml` — Placeholder (unused)
